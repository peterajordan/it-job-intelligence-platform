# AI Agents

This folder contains the conceptual workflow and future modular structure for the AI-assisted job intelligence platform.

## Current Agent Architecture

### 1. Job Retrieval Agent
Responsible for:
> Querying external job APIs,
> Retrieving live job opportunities,
> Processing API responses.

### 2. Filtering and Classification Agent
Responsible for:
> Keyword filtering,
> Remote opportunity identification,
> Duplicate detection,
> Opportunity categorization.

### 3. Opportunity Scoring Agent
Responsible for:
> Fit score calculations,
> Priority ranking,
> Relevance analysis.

## Planned Future Agents

- Resume Matching Agent
- Interview Preparation Agent
- Skill Gap Analysis Agent
- Follow-up Automation Agent

## Future Direction

The long-term goal is to evolve the platform into a modular AI-assisted workflow system capable of automating and optimizing large portions of the IT job search process.
